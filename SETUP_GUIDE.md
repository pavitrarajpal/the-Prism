# Project Setup Guide: The Prism

Follow these steps to connect your project to GitHub and Supabase.

## 1. GitHub Setup

1.  Go to [github.com/new](https://github.com/new).
2.  **Repository name**: `antigravity-prism` (or any name you prefer).
3.  **Visibility**: Choose Public or Private.
4.  **Initialize**: Do **NOT** check "Add a README file", "Add .gitignore", or "Choose a license". We want an empty repository.
5.  Click **Create repository**.
6.  Copy the HTTPS URL (e.g., `https://github.com/your-username/antigravity-prism.git`).

**Action required**: Paste the GitHub URL in the chat.

## 2. Supabase Setup

1.  Go to [supabase.com/dashboard/projects](https://supabase.com/dashboard/projects) and click **New project**.
2.  **Organization**: Select your organization.
3.  **Name**: `Antigravity Prism`.
4.  **Database Password**: Generate a strong password and save it in a password manager.
5.  **Region**: Choose a region close to you (e.g., Mumbai, Singapore).
6.  Click **Create new project**.
7.  Wait for the project to provision.
8.  Go to **Project Settings** (cog icon) -> **API**.
9.  Copy the **Project URL** and the **anon public key**.

**Action required**: Paste the **Project URL** and **anon public key** in the chat.
